The following Dublin Core metadata can be set for every document:

    * Lang
    * Title
    * Subject
    * Description
    * Publisher

In addition MKDoc automatically generates the following Dublin Core metadata for documents:

    * Identifier
    * Creator
    * Contributor
    * Date Created
    * Date Modified
    * References
    * Replaces
