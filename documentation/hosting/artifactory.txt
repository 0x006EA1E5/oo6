Do a simple install

Add in jboss.org repo (for jdocbook plugin)

  <remoteRepository>
            <key>jboss.org</key>
            <description>jboss.org</description>
            <handleReleases>true</handleReleases>
            <handleSnapshots>true</handleSnapshots>
            <excludesPattern>org/artifactory/**,org/jfrog/**</excludesPattern>
            <url>http://repository.jboss.org/maven2</url>
   </remoteRepository>

Add in our unreolvable dependencies (note add in otherojects-repo to settings.xml with admin user/pass):
                    
mvn deploy:deploy-file -DrepositoryId=otherojects-repo -Durl=http://maven.richaston.com:8081/artifactory/ext-releases -Dfile=flexjson.jar -DpomFile=pom.xml
mvn deploy:deploy-file -DrepositoryId=otherojects-repo -Durl=http://maven.richaston.com:8081/artifactory/ext-releases -Dfile=flexjson-src.jar -DpomFile=pom.xml -Dclassifier=sources
