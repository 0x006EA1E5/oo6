In order to do multiple resource transactions (XA) you need a JTA implementation.
All app servers have their JTA implementations and there are a few standalone ones:

Bitronix Transaction Manager (BTM) (LGPL):
http://www.bitronix.be/Btm/Overview
http://docs.codehaus.org/display/BTM/Home

Atomikos (Apache license v2):
http://www.atomikos.com/products.html

Jencks (Apache license v2):
http://jencks.codehaus.org/

JOTM (BSD-style license):
http://jotm.objectweb.org/

SimpleJTA (Apache license v2):
http://sourceforge.net/projects/simplejta
