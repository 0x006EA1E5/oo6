Setting up a clustered OTHERobjects site with Terracotta
--------------------------------------------------------

#) Download terracotta 2.4

#) Start terracotta server (which coordinates the distribution of shared objects)
   with default config (to change the log and data paths copy sample/tc-config.xml
   to the directory from where you start tc and customise).
   
   ./bin/start-tc-server.sh

#) Modify OO sample tc-config.xml file making sure to get jee-application name right.

#) Build site war file and copy to tomcat/webapps

#) Start a Tomcat instance using Terracotta using our start-tc-tomcat script.
